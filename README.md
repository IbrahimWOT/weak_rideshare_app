# 🚗 CityRide - Weak Rideshare App (Cyber Security Lab)

**weak_rideshare_app** হল একটি ইচ্ছা করে দুর্বল করে তৈরি করা (Intentionally Vulnerable) পিএইচপি এবং মাইএসকিউএল ওয়েব অ্যাপ্লিকেশন। এটি মূলত সাইবার সিকিউরিটি এবং ওয়েব পেন্ডুলেশন টেস্টিং প্র্যাকটিসের জন্য ডিজাইন করা হয়েছে, যাতে শিক্ষার্থীরা বাস্তবসম্মত পরিবেশে কুয়েরি ব্রেকিং এবং অটোমেটেড টুলস টেস্ট করতে পারে।

## 🎯 Key Features & Lab Coverages
- **Realistic Rideshare UI:** রাইডার এবং ড্রাইভারদের জন্য আলাদা ডেডিকেটেড পোর্টাল।
- **Role-Based Backend Architecture:** `users` টেবিলের ভেতর রাইডার এবং ড্রাইভারদের ডেটা পৃথক করার জন্য রোল-বেসড মেকানিজম।
- **SQL Injection Laboratory:** লগইন ফর্মটি ইনপুট স্যানিটাইজেশন ছাড়াই তৈরি করা হয়েছে, যা সরাসরি **SQLmap** এবং ম্যানুয়াল পেলোড অ্যাটাকের জন্য উন্মুক্ত।
